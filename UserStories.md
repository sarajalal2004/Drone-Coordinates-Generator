# 👥 User Stories
The main user stories

## 👤 User Authentication & Profile

### Registration & Authentication
- As a new user, I want to register an account so that I can access the system.
- As a registered user, I want to verify my account using an email token so that my account becomes active.
- As a user, I want to log in securely so that I can access protected resources.
- As a user, I want to change my password so that I can maintain account security.
- As a user, I want to reset my password using a recovery token so that I can regain access if I forget it.

### Profile Management
- As a user, I want to update my profile information so that my account details remain accurate.
- As a user, I want to upload or update my profile picture so that my profile is personalized.

### Administration
- As an admin, I want to deactivate user accounts so that I can restrict access when necessary.
- As an admin, I want to reactivate user accounts so that authorized users can regain access.
- As an admin, I want to update user roles so that permissions are managed correctly.

---

## 🚁 Drone Management

### Drone Registration
- As a user, I want to add a drone to the system so that it can be managed and utilized for operations.

### Drone Information
- As a user, I want to view all registered drones so that I can see available resources.
- As a user, I want to view detailed information about a specific drone so that I can understand its capabilities.

### Drone Maintenance
- As a drone owner or admin, I want to update drone information so that the data remains accurate.
- As a drone owner or admin, I want to delete a drone so that obsolete drones can be removed from the system.

### Drone Rental 
- As a drone owner or admin, I want to manage drone rental so that I can control drone usage.

---

## 🖼️ Drone Image Processing

### Image Upload & Analysis
- As a user, I want to upload an image so that the system can analyze it.
- As a user, I want the system to generate optimal coordinates from an uploaded image so that drones can be positioned accurately.

### Coordinate Generation
- As a user, I want the system to identify the best target points within an image so that drone deployment is effective.
- As a user, I want the system to utilize all available drones efficiently when generating coordinates so that resources are maximized.

### Automated Processing
- As a user, I want image processing to be automated so that manual coordinate calculation is not required.
- As a user, I want processing results to be stored for future reference and analysis.

---

## 📜 History Tracking

### Personal History
- As a user, I want to view my own processing history so that I can track previous operations.
- As a user, I want to view details of a specific history record so that I can review processing results.

### Administrative History
- As an admin, I want to view all history records so that I can monitor system activity.
- As an admin, I want to delete history records when necessary so that outdated information can be managed.
